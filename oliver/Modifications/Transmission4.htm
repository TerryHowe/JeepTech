As is turned out, the tip of the T-18 output shaft was bigger than the tip of the T-98 shaft as was the corresponding hole in the input shaft.  The T-98 rollers were larger, while the T-18 rollers were were the same size as the cluster gear rollers, which was probably the reason for the change.  The Ford T-18 output shaft wouldn't work with my transfercase so I had to find another shaft.  Luckily, I had a friend who had an extra T-18 shaft out of a Scout which would work with both the Ford input shaft and the Dana 18 transfercase, but there were still a few more differences.
