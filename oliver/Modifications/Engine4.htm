Placing the Engine so tight against the frame rail created some other problems.  The Ford power steering pump hit the frame rail and would have been right in the way of the planned position of the Scout power steering gear box.  I will have to mount the pump somewhere else.  More on that in the steering section.  This wouldn't have been a problem with the original steering set up.  However, you can also see the original steering box interferes with the oil cooler and filter.  This was a problem with the pinto engine as well, and was solved by using a remote oil filter.  When the Ross steering box is removed, I will have enough room for the oil cooler, but will still have to remote mount the oil filter.  I still have to wire the engine computer and install an electric fuel pump.
